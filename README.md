\# Proyecto de trabajo de unidad para Fundamentos de Programación



\## UNIBIBLIOAPP - GESTIÓN DE BIBLIOTECA



\### Descripción del Proyecto

UniBiblioApp es nuestra propuesta de solución desarrollada en Dev-C++ con el objetivo de resolver el desorden en el préstamo de laptops y libros dentro de la biblioteca universitaria. El sistema permite registrar estudiantes, mapear en tiempo real su ubicación (mesas y sillas) y ver el inventario usado.
